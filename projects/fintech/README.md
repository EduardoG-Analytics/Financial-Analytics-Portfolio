## Overview

This project presents an end-to-end Business Intelligence and Financial Analytics solution developed as part of a technical business case for a lending company.

The platform consolidates operational and financial information to support lending decisions, portfolio monitoring, borrowing capacity analysis and executive reporting through an interactive web application.

## Business Context

Financial institutions require reliable and timely information to monitor lending portfolios, evaluate borrowing capacity and support credit-related decision-making.

Traditional spreadsheet-based reporting can limit visibility into portfolio performance, making it difficult to identify risks, monitor collections and evaluate business opportunities.

This solution centralizes financial and operational information into an interactive analytics platform designed to support business users through intuitive visualizations and automated calculations.

---

## Business Problem

Lending organizations must continuously monitor portfolio performance while ensuring that lending decisions are supported by accurate and up-to-date financial information.

The challenge is to consolidate information from multiple operational sources into a single environment capable of providing:

- Portfolio visibility.
- Borrowing Base calculations.
- Customer-level performance.
- Collection monitoring.
- Executive KPIs.
- Financial forecasting.

## Solution Architecture

The platform follows an end-to-end analytics workflow designed to transform operational lending data into actionable financial insights.

```text
Operational Data Sources
(Loans, Customers, Payments, Collections)
                    │
                    ▼
         Data Cleaning & Validation
                    │
                    ▼
        Business Rules Implementation
                    │
                    ▼
      Financial Calculations & KPIs
(Borrowing Base, Delinquency, Portfolio)
                    │
                    ▼
      Interactive Streamlit Application
                    │
                    ▼
 Executive Reporting & Decision Support
```

The architecture integrates data transformation, financial calculations and business logic into a single analytics platform designed for operational and executive users.

## Data Sources

The analytics platform is built upon lending portfolio data containing operational and financial information for each loan.

Key data attributes include:

| Field | Description |
|------|-------------|
| Loan ID | Unique loan identifier |
| Principal Balance | Outstanding loan balance |
| Recurrency | Payment recurrence information |
| Status | Current loan status |
| Shipping Date | Loan operational date |
| Shared Report Date | Reporting reference date |
| Ineligible Collateral Loans | Collateral eligibility indicator |
| MT Status | Portfolio operational status |

The dataset was transformed to support business rules, KPI calculations and interactive portfolio analysis.
